XRAT - What is it?

In simple terms, an XSS Shell.

Caution!

XRAT is intended to be a security-testing and educational tool. It can be used for malicious and illegal purposes but I do not condone such activities and will take no such responsibility for malicious activities carried out using the tool. Use at your own peril. If you are going to use the job feature for a distributed grid computing job, make sure that the individual nodes are either your computers or you have consent to use that users CPU cycles.

XRAT stands for XSS Remote Admin Tool. So basically it provides you with a shell to control hooked browsers. A hooked browser is a user's browser that has loaded the client-side javascript, and is now communicating with this server and exposing all of its functionality to be controlled. In addition to this, XRAT attempts to provide a proof-of-concept for Javascript-based Grid Computing.

Inside XRAT is the ability to take over a user's browser and use one of the preinstalled modules to carry out some functionality including keylogging, stealing cookies, navigating to other pages, through the use of persistent iframes.
